# Portafolio CV - Emmanuel Salas Ovalle

Portafolio profesional y CV web de Emmanuel Salas Ovalle, Ingeniero en Desarrollo y Gestión de Software.

## Tecnologías

- HTML5
- CSS3
- JavaScript
- Font Awesome 6

## Secciones

- **Inicio** — Presentación principal
- **Sobre Mi** — Información personal, formación académica y descarga de CV
- **Servicios** — Desarrollo web, SEO, docencia, bases de datos y consultoría
- **Habilidades** — Barras de progreso animadas con nivel de dominio
- **Portfolio** — Galería de proyectos realizados
- **Contacto** — Formulario de contacto vía Formspree
